Intrinsic Ratios for LESS and SASS

A reusable class to be used in a dynamic stylesheet language (LESS or SASS). Includes comments.

SCSS/LESS Example

.intrinsicContainer {
     > .intrinsicWrapper {
          height: 0;
          position: relative;
          > .intrinsicElement {
               left: 0;
               position: absolute;
               top: 0;
               width: 100%;
               height: 100%;
          }
     }
}

HTML Code Example

<div class="intrinsicContainer">
     <div class="intrinsicWrapper">
          <div class="intrinsicElement imageExample">
               <!-- THIS AREA IS NOW 4x3 -->
          </div>
     </div>
</div>
